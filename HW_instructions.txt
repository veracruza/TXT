TXT
1. Создать внешний репозиторий c названием TXT.
 	1) В Github  NEW >>Repository name: "TXT".(+ Public,+Add a Readme file.) Create repository. 

2. Клонировать репозиторий TXT на локальный компьютер. 
 	1)Code>HTTPS>скопировать.
 	2)Открыть папку. mkdir group_29_free
 	3)Клонировать репозиторий: git clone https://github.com/veracruza/TXT.git 
(Вставляем скопированную ранее ссылку).
 
3. Внутри локального TXT создать файл “GIT.txt     
	1) touch GIT.txt
4. Добавить файл под гит. 
	1) git add GIT.txt 
5. Закоммитить файл. 
	1)git commit -m "create GIT.txt"
6. Отправить файл на внешний GitHub репозиторий. 
	1)git push
7. Отредактировать содержание файла “GIT.txt” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате TXT.
	1) vim GIT.txt
8. Отправить изменения на внешний репозиторий. 
	1) git commit -am "info"
	2) git push
9. Создать файл preferences.txt 
	1) touch preferences.txt
10. В файл preferences.txt” добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате TXT.
	1) vim preferences.txt
11. Создать файл sklls.txt добавить информацию о скиллах которые будут изучены на курсе в формате TXT 						     
	1)cat >> sklls.txt
12. Сделать коммит в одну строку. 
	1) git add . | git commit -m "changes"
 13. Отправить сразу 2 файла на внешний репозиторий. 
	1) git push
 14. На веб интерфейсе создать файл bug_report.txt. 
	1) Add file->Create new file
 15. Сделать Commit changes (сохранить) изменения на веб интерфейсе. Commit new file
 16. На веб интерфейсе модифицировать файл bug_report.txt, добавить баг репорт в формате TXT. 
	1) Edit this file.
 17. Сделать Commit changes (сохранить) изменения на веб интерфейсе.
	1) git fetch (посмотреть изменения)
	2) git pull (скопировать все на локальный репозиторий)
